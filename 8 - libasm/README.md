#### Build and Run:

* Build mandatory part:<br/>
```> make```

* Build bonus part:<br/>
```> make bonus```


#### Build and Run tests:

* Test mandatory part:<br/>
```> make test```

* Test bonus part:<br/>
```> make btest```
